Features
🤖

multi-agent

Codex, Claude, OpenCode, Pi, and Droid — all in one app. Switch between them per session, or run several in parallel.

📡

auto-discovery

Finds agents running on your local network automatically. No IP addresses, no manual configuration.

🐈

alleycat

Peer-to-peer pairing via QR code. Connect through NATs and corporate firewalls — no VPN, no public IP, no port forwarding.

🔑

ssh connections

Connect to any remote machine over SSH. Litter starts your agent on the server for you.

🎙

voice input

Real-time voice with your agent — transcription and conversation happen seamlessly while you walk, drive, or think.

🎨

70+ themes

Dozens of editor themes — Catppuccin, Dracula, Nord, Gruvbox, Solarized, and more. Dark and light.

How it works

three steps, zero config

1

find your server

Auto-discovery on your local network, alleycat QR pairing for anywhere else, or paste an SSH address.

2

pick an agent

Codex, Claude, OpenCode, Pi, or Droid. Litter starts the agent on your server for you.

3

get to work

Pick up exactly where you left off. Sessions, threads, and context follow you everywhere.
